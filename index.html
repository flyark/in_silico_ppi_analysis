FlyPredictome Analysis

Paste a FlyPredictome URL to automatically fetch interaction data, extract LIR/cLIR residue indices, and generate a ChimeraX .cxc visualization script.

Universal Prediction Analysis

Upload prediction files from AlphaFold3, AlphaFold2, ColabFold, Boltz, Chai-1, OpenFold, or RoseTTAFold. Calculate AFM-LIS metrics (iLIS, LIS, cLIS), view PAE/LIS/cLIS maps, and generate ChimeraX scripts.

AlphaFold DB Dimer Analysis

Search the AlphaFold Protein Structure Database for dimer predictions by UniProt ID, calculate iLIS between chains, view PAE/LIS/cLIS maps, and generate ChimeraX scripts.

AlphaFold DB Monomer Subdomain Analysis

Fetch monomer predictions from the AlphaFold Protein Structure Database by UniProt ID. Auto-detect domains, view PAE/LIS/cLIS maps, and calculate intramolecular iLIS between domains.

Step-by-Step Guide

Visual instructions for all tools — from getting your data to opening the visualization in ChimeraX. Includes metric definitions and color scheme references.
