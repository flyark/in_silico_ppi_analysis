FlyPredictome → ChimeraX

Paste a FlyPredictome URL to automatically fetch interaction data, extract LIR/cLIR residue indices, and generate a ChimeraX .cxc visualization script. Supports all FlyPredictome prediction pairs.
🧬

AlphaFold3 → ChimeraX

Upload an AlphaFold3 zip file to calculate AFM-LIS metrics (iLIS, LIS, cLIS, LIR, cLIR), view PAE/LIS/cLIS maps, score matrices, and generate ChimeraX scripts. Supports multi-chain complexes.
📖

Step-by-Step Guide

Visual instructions for both tools — from getting your data to opening the visualization in ChimeraX. Includes metric definitions and color scheme references.
