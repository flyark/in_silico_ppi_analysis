Quick Start

  1. Install UCSF ChimeraX
  2. Run a prediction on AlphaFold Server
  3. Download the results as a zip file
  4. Drop the zip file above and click Process
  5. Explore PAE maps, score matrices, and LIS/cLIS maps
  6. Click a chain pair in the table to generate a ChimeraX script
  7. Download .cxc + .cif, place in the same folder
  8. Open the .cxc file to open ChimeraX

Metrics (AFM-LIS)

LIRLocal Interaction Residues (PAE ≤ 12 Å)
cLIRcontact-filtered LIR (PAE ≤ 12 Å & Cβ ≤ 8 Å)
LISLocal Interaction Score — normalized PAE confidence (0–1)
cLIScontact-filtered LIS — restricted to direct contacts
iLISintegrated LIS — √(LIS × cLIS)
ipTMinterface predicted TM-score
