How to Use  (Full guide)

Quick Start

  1. Install UCSF ChimeraX
  2. Enter a UniProt ID (e.g., P28482) and click Fetch
  3. Define two or more regions of interest by residue range
  4. Click Calculate to compute interaction metrics
  5. Review the PAE map, iLIS scores, and region pair table
  6. Download the .cxc + .pdb files, place them in the same folder, then double-click the .cxc — ChimeraX will launch automatically

Metrics (AFM-LIS)

iLISintegrated LIS — √(LIS × cLIS)
LISLocal Interaction Score — normalized PAE confidence (0–1)
cLIScontact-filtered LIS — restricted to direct contacts
LIRLocal Interaction Residues (PAE ≤ 12 Å)
cLIRcontact-filtered LIR (PAE ≤ 12 Å & Cβ ≤ 8 Å)
