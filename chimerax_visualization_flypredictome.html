Quick Start

  1. Install UCSF ChimeraX
  2. Go to FlyPredictome and search for a protein pair
  3. Copy the FlyPredictome result page URL
  4. Paste the URL above and click Generate
  5. Download the .cxc script and .pdb file
  6. Place both files in the same folder
  7. Open the .cxc file to open ChimeraX

Metrics (AFM-LIS)

LIRLocal Interaction Residues — residues in the confident interaction region (PAE ≤ 12 Å)
cLIRcontact-filtered LIR — residues in direct physical contact (PAE ≤ 12 Å & Cβ ≤ 8 Å)
LISLocal Interaction Score — normalized PAE confidence across the interface (0–1)
cLIScontact-filtered LIS — LIS restricted to direct physical contacts
iLISintegrated LIS — √(LIS × cLIS), geometric mean of LIS and cLIS for a balanced score (confident domain & confident contact)
ipTMinterface predicted TM-score — AlphaFold-Multimer's global interface confidence
