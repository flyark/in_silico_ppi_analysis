Quick Start

  1. Enter a UniProt ID (e.g., P28482) and click Fetch
  2. Define two or more regions of interest by residue range
  3. Click Calculate to compute interaction metrics
  4. Review the PAE map, iLIS scores, and region pair table
  5. Download the .cxc + .pdb files
  6. Open the .cxc file in UCSF ChimeraX

Metrics (AFM-LIS)

LIRLocal Interaction Residues (PAE ≤ cutoff)
cLIRcontact-filtered LIR (PAE ≤ cutoff & Cβ ≤ cutoff)
LISLocal Interaction Score — normalized PAE confidence (0–1)
cLIScontact-filtered LIS — restricted to direct contacts
iLISintegrated LIS — √(LIS × cLIS)
