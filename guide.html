ChimeraX Visualization for LIS — Guide

Analyze AlphaFold-predicted protein interactions and generate ChimeraX visualization scripts

Home FlyPredictome-ChimeraX AlphaFold3-ChimeraX GitHub

Prerequisites

  • Install UCSF ChimeraX (free for academic use)
  • A modern web browser (tested in Google Chrome; folder selection requires Chrome)
  • No Python or other software needed — all analysis runs locally in your browser (no data is uploaded to any server)

FlyPredictome → ChimeraX

1

Go to FlyPredictome and find your protein pair

Visit flyrnai.org/tools/fly_predictome, search for a protein pair, and open the prediction result page.

Copy the page URL from your browser's address bar.

https://www.flyrnai.org/tools/fly_predictome/web/prediction/Akt1/foxo/SET_47/
2

Paste the URL and click Generate

Open FlyPredictome-ChimeraX, paste the URL, choose a rank (1–5), and click Generate. The tool fetches data automatically (may take a few seconds).

Download both the .cxc script and .pdb structure file. Place them in the same folder, then double-click the .cxc file to open ChimeraX.

Both files must be in the same folder. The .cxc script references the .pdb by filename.

AlphaFold3 → ChimeraX

1

Get your AF3 prediction zip

Run a prediction on AlphaFold Server and download the results as a zip file. The zip should contain *_full_data_*.json, *_summary_confidences_*.json, and *_model_*.cif files.

fold_my_prediction.zip
├── ..._full_data_0.json  ← PAE matrix
├── ..._summary_confidences_0.json  ← ipTM
├── ..._model_0.cif  ← 3D structure
└── (× 5 models)
2

Drop the zip file and click Process

Open AlphaFold3-ChimeraX, drag-and-drop your zip file (or click to browse), then click Process. Processing takes several seconds per model.

Drop zip here

Optionally adjust PAE cutoff (default: 12 Å) and Cβ cutoff (default: 8 Å) before processing.
3

Explore PAE maps and score matrices

The tool generates per-model visualizations:

  • PAE Maps — blue (confident) to red (uncertain)
  • Score Matrix — iLIS (Oranges, lower-left) / ipTM (Purples, upper-right)
  • Residue Count Matrix — LIR (Blues) / cLIR (Greens)
  • LIS Maps — per-residue interaction scores (Blues)
  • cLIS Maps — contact-filtered scores (Greens)

Select a chain pair from the table

5

Download and open in ChimeraX

Three download options:

  • Download .cxc — script for the selected chain pair
  • Download .cif — structure file for the selected model
  • Download All Chains .cxc — single script showing all chains, colored by chain
chainA_chainB_interface.cxc + fold_..._model_0.cif → Same folder → Open .cxc
For multi-chain complexes, use "Download All Chains .cxc" to see the full complex with each chain in a different color. Uses ChimeraX's color bychain command.

Color Scheme Guide

🎨

Default: Blue / Orange gradient

LIR regions (confident interaction domain, PAE ≤ 12 Å) are shown in light colors. cLIR residues (direct contact interface, PAE ≤ 12 Å & Cβ ≤ 8 Å) are highlighted in darker colors. Non-interacting regions are hidden.

Chain A — LIR (light blue)
Chain A — cLIR (dark blue)
Chain B — LIR (light orange)
Chain B — cLIR (dark orange)
⚙

Preset categories

  • Gradient — light color for LIR, dark color for cLIR (e.g., Blue/Orange, Purple/Gold, Teal/Coral)
  • Solid — same color for both LIR and cLIR, useful when you don't need to distinguish the interface boundary
  • ChimeraX defaults — familiar colors for ChimeraX users (Cornflower/Tomato, SkyBlue/Gold, etc.)
  • Multi-chain (tab10) — auto-assigned distinct colors per chain, shown only for complexes with more than 2 chains

Swap A ↔ B button reverses chain colors without changing the color combination.

↻

Applying color changes

After selecting a new color scheme or adjusting individual colors:

  • FlyPredictome: Click Generate again to regenerate the .cxc script with updated colors
  • AlphaFold3: Click a chain pair row in the table again, or click Process to reprocess with updated colors and cutoffs

The PAE/LIS/cLIS maps are not affected by color choices — they use fixed colorscales (bwr, Blues, Greens). Only the ChimeraX .cxc script uses the selected chain colors.

AFM-LIS Metrics

MetricFull NameDefinition
LIRLocal Interaction ResiduesResidues in the confident interaction region (PAE ≤ 12 Å)
cLIRcontact-filtered LIRResidues in direct physical contact (PAE ≤ 12 Å & Cβ ≤ 8 Å)
LIALocal Interaction AreaConfident interaction area (PAE ≤ 12 Å)
cLIAcontact-filtered LIAConfident interaction area within contact distance (PAE ≤ 12 Å & Cβ ≤ 8 Å)
LISLocal Interaction ScoreAverage of inversely scaled PAE (0–1, higher is better) within LIA
cLIScontact-filtered LISAverage of inversely scaled PAE within cLIA (contact-filtered)
iLISintegrated LIS√(LIS × cLIS) — geometric mean capturing both rigid and transient interactions
ipTMinterface predicted TM-scoreAlphaFold's global interface structural confidence metric

PAE transformation: Raw PAE values (0–30 Å) are linearly scaled to confidence scores: PAE = 0 → score = 1 (highest confidence), PAE = cutoff → score = 0, PAE > cutoff → 0. Default PAE cutoff is 12 Å.

Contact map: Built from Cβ atom distances (Cα for glycine). Two residues are in contact if their Cβ–Cβ distance ≤ 8 Å. For nucleic acids, phosphorus (P) atoms are used with a 4 Å distance adjustment.

Symmetrization: Results for chain pairs (A→B) and (B→A) are averaged. LIR/cLIR residue sets are the union across all models.

Suggested threshold: iLIS ≥ 0.223 was established at 10% FDR using large-scale Y2H reference datasets in yeast, fly, and human (see Kim et al., 2025 supplementary text for details). This threshold provides a useful starting point for identifying high-confidence interactions.

Caution: Computational predictions should be interpreted with care. The iLIS threshold was derived from specific reference datasets and may not be optimal for all biological contexts. Optimal thresholds may vary depending on the species, protein types, and experimental system. Experimental validation is highly recommended before drawing biological conclusions from predicted interactions.
